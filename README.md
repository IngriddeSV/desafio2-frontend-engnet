# Processo Trainee EngNet - Desafio 2 Front-End

## O Proposto :

Este desafio representa o primeiro contato prático com o desenvolvimento front-end moderno dentro do processo trainee. Mais do que apenas escrever código, um bom desenvolvedor front-end precisa ser capaz de transformar interfaces planejadas por designers em páginas reais, funcionais e visualmente fiéis ao projeto original.

No mercado, é muito comum receber layouts prontos em ferramentas como e convertê-los em aplicações web responsivas. Por isso, neste desafio vocês irão trabalhar exatamente essa habilidade, utilizando , uma das tecnologias mais utilizadas atualmente para estilização moderna e produtiva. O objetivo é desenvolver atenção aos detalhes, organização visual, domínio de layout e capacidade de interpretar interfaces profissionais.

O desafio consistiu em reproduzir uma Landing Page a partir de um design pronto no Figma, realizando a inspeção técnica dos elementos e desenvolvendo uma página o mais fiel possível ao layout original, aplicando corretamente:

- Cores utilizadas (hexadecimal ou RGB);
- Tipografia (família, peso e tamanho);
- Espaçamentos internos e externos (padding e margin);
- Bordas e arredondamentos;
- Ícones e imagens exportados do Figma;
- Estrutura e comportamento responsivo.

## Como Executar o Projeto Localmente

1. **Clonar o repositório:** Abra o seu terminal e rode o comando abaixo para puxar os arquivos:

   git clone [https://github.com/SEU_USUARIO/SEU_REPOSITORIO.git](https://github.com/SEU_USUARIO/SEU_REPOSITORIO.git)

2. **Entrar na pasta:**

   cd SEU_REPOSITORIO
   Abrir o projeto:

3. Se você usa o VS Code, o jeito mais fácil é abrir a pasta nele e usar a extensão Live Server clicando em "Go Live" (assim o Tailwind e os scripts rodam certinho com um servidor local simulado).

4. Se preferir algo mais direto, é só ir na pasta pelo gerenciador de arquivos do seu computador e clicar duas vezes no index.html para abrir direto em qualquer navegador (Chrome, Edge, Firefox, etc.).
